Animation with @keyframes :

In this example, we see a simple animation made with CSS; with @keyframes more specifically.
For this type of animation we must detail the "keyframes" or percentages of the animation itself.

Results:

Code:

#iso .st2{ transition: 1s; animation: 3s isoAnima infinite; animation-delay: 2500ms; }
@keyframes isoAnima { 0%{transform: translate(0,0);} 40%{ transform: translate(0,-12px); opacity: 1; filter: blur(0px) } 42%,44%{ opacity: 0.3; filter: blur(0px);} 43%,45%{ opacity: 0.75; filter: blur(10px);} 47%{ opacity: 1; filter: blur(0px)} 90%{ transform: translate(0,3px); } 100%{ transform: translate(0,0); } }
